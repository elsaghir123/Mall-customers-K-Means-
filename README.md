## Dataset

The dataset used in this project is the **Mall Customers Dataset**, available on Kaggle:

🔗 https://www.kaggle.com/datasets/kandij/mall-customers/data

---

## About the Dataset

This dataset contains basic information about customers of a supermarket mall collected through membership cards.

The data includes demographic information and customer behavior that can be used for **customer segmentation analysis**.

Customer segmentation helps businesses understand different groups of customers and design targeted marketing strategies.
